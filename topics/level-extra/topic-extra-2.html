📦 ক্লাস (Class)

ক্লাস মানে হলো একটি ব্লুপ্রিন্ট বা নকশা — যার সাহায্যে একই ধরনের অনেকগুলো জিনিস বানানো যায়।
🏭📋
📖 গল্প: গাড়ির কারখানা
গাড়ির কারখানায় একটি নকশা (ব্লুপ্রিন্ট) থাকে। সেই নকশা দেখে হাজার হাজার গাড়ি বানানো হয়।

🚗 নকশা = ক্লাস
🚗 নকশা থেকে বানানো আসল গাড়ি = অবজেক্ট (Object)

প্রতিটি গাড়ির রং, মডেল আলাদা হতে পারে, কিন্তু সব গাড়ি একই নকশা থেকে বানানো!
📌 ক্লাস = নকশা / ব্লুপ্রিন্ট | অবজেক্ট = আসল জিনিস
সহজ ভাষায়

ক্লাস (Class) = একটি ছাঁচ (ঢালাই করার জন্য)

অবজেক্ট (Object) = সেই ছাঁচ থেকে তৈরি আসল জিনিস

যেমন: একটি কুকুরের জাত হলো ক্লাস। আর তোমার পোষা কুকুর হলো অবজেক্ট!

কোড উদাহরণ (ধারণার জন্য) 
class Car {
  constructor(color, speed) {
    this.color = color;
    this.speed = speed;
  }
  
  start() {
    return "গাড়ি চালু হয়েছে! 🚗";
  }
}

const myCar = new Car("লাল", 100);
console.log(myCar.color);  // লাল
console.log(myCar.start()); // গাড়ি চালু হয়েছে!
