🎯 শর্ত (Condition)

শর্ত হলো সিদ্ধান্ত নেওয়ার নিয়ম — যদি কোনো কিছু সত্য হয়, তাহলে একটি কাজ করো, না হলে অন্য কাজ করো।
প্রোগ্রামিং-এ if-else দিয়ে আমরা শর্ত ব্যবহার করি।
গণিতের ধারণা
ক্লাস ৪-৫ এ আমরা শিখি — "যদি লাভ হয় তবে হাসো, ক্ষতি হলে কাঁদো" — এটাই শর্ত!
প্রোগ্রামিং-এ শর্ত: if (বৃষ্টি হয়) { আমিব যাব } else { যাব না }
গল্প ১: কাকন ও আমল
🐧
কাকন
🐻
আমল
কাকন ও আমল দুই খুব ভালো বন্ধু। তারা খেলতে খুব ভালোবাসে। কিন্তু আমলের একটা শর্ত আছে —
"আমল শুধু তখনই খেলতে যায়, যখন কাকন খেলতে যায়।"
যদি (কাকন খেলতে যায়) { আমল খেলতে যাবে } না হলে { আমল খেলতে যাবে না }
📝 প্রশ্ন: কাকন খেলতে না গেলে, আমল কী করবে?
গল্প ২: বৃষ্টি ও খেলা
☔
বৃষ্টি
👦
আমি
আমি মাঠে খেলতে যেতে চাই। কিন্তু আমার মা একটি শর্ত দিয়ে দিয়েছেন —
"বৃষ্টি পড়লে খেলতে যাবি না, বৃষ্টি না পড়লে যাবি।"
যদি (বৃষ্টি পড়ে) { আমি খেলতে যাব না } না হলে { আমি খেলতে যাব }
📝 প্রশ্ন: বৃষ্টি পড়লে আমি কী করব?
গল্প ৩: তুমি কথা বললে আমি বলব
🗣️
তুমি
🤖
আমি (রোবট)
আমার একটি মজার রোবট আছে। রোবটটির একটি শর্ত আছে —
"তুমি কথা বললে, আমিও কথা বলব। তুমি না বললে, আমিও চুপ থাকব।"
যদি (তুমি কথা বলো) { রোবট কথা বলবে } না হলে { রোবট চুপ থাকবে }
📝 প্রশ্ন: তুমি কথা না বললে রোবট কী করবে?
গেম: শর্ত খেলা
🎲 বাইরে বৃষ্টি হচ্ছে?
যদি (বৃষ্টি হয়) { ঘরে থাকো } না হলে { বাইরে খেলো }

২. "যদি বৃষ্টি হয়, তবে ছাতা নিবো" — এখানে শর্তটি কী?
