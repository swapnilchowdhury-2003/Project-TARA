🎯 If/Else - যদি/তাহলে/নাহলে

If/Else হলো প্রোগ্রামিং-এর সবচেয়ে গুরুত্বপূর্ণ সিদ্ধান্ত নেওয়ার নিয়ম।
যদি একটি শর্ত সত্য হয়, তাহলে একটি কাজ করো। না হলে (else) অন্য কাজ করো।
If/Else — শর্ত সত্য হলে এক কাজ, মিথ্যা হলে অন্য কাজ
গল্প: নন্দির কেনাকাটা
নন্দি দোকানে গেলো। তার কাছে কিছু টাকা আছে। দোকানদার বলল —
"যদি তোমার ১০ টাকার বেশি থাকে, তাহলে তুমি চিপস কিনতে পারবে।
না হলে (else), তুমি ক্যান্ডি কিনতে পারবে।"
if (টাকা > ১০) {
    কিনবে = "চিপস 🍟";
} else {
    কিনবে = "ক্যান্ডি 🍬";
}
খেলা: নন্দির কেনাকাটা

If/Else কীভাবে কাজ করে?

if (শর্ত) { → যদি শর্তটি সত্য হয়
    // সত্য হলে এই কাজ করো
} else { → যদি শর্তটি মিথ্যা হয়
    // মিথ্যা হলে এই কাজ করো
}

📌 উদাহরণ: if (টাকা > ১০) { চিপস } else { ক্যান্ডি }

🔹 যদি টাকা ১০-এর বেশি হয় → চিপস কিনবে
🔹 যদি টাকা ১০ বা তার কম হয় → ক্যান্ডি কিনবে

If/Else সম্পর্কে জ্ঞান পরীক্ষা
১. if-else ব্যবহার করা হয় কেন?
২. if (টাকা > ১০) { চিপস } else { ক্যান্ডি } — টাকা = ৮ হলে কী হবে?
৩. else অংশটি কী কাজ করে?
৪. নন্দির কাছে ১৫ টাকা থাকলে সে কী কিনবে?
৫. if (বৃষ্টি == true) { ছাতা নিবো } else { ছাতা নিবো না } — এখানে শর্তটি কী?

If/Else শিখে ফেলেছো? পরবর্তী টপিকে যাও 👆
